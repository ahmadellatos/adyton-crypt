Thank you for installing Adyton Crypt!

Adyton Crypt protects your files with AES-256-GCM authenticated encryption and Argon2id key derivation — a slow, memory-hard function built to resist password guessing. (Vaults created by older versions remain readable.)

IMPORTANT NOTES:
- Your files are encrypted locally on your device. No data is sent to any server.
- Always remember your password. There is no password recovery.
- We recommend testing encryption/decryption with non-critical files first.

For more information, updates, or support:
- GitHub: https://github.com/ahmadellatos/adyton-crypt
- Report issues or suggestions on our repository.

Stay safe and encrypt responsibly.

— Adyton Security Team
